# Mobile Popup Viewer

**크롬에서 작은 모바일 형태의 팝업 창으로 웹사이트를 빠르게 열 수 있는 확장 프로그램입니다.**

---

## 개요

이 확장 프로그램은 웹사이트를 작은 팝업 창으로 열어  
데스크탑 환경에서도 모바일처럼 간편하게 브라우징할 수 있도록 도와줍니다.

탭 전환 없이 빠르게 확인하거나, 작업 중 보조 화면으로 활용하기에 적합합니다.

---

## 주요 기능

- **400x700 크기의 팝업 창 제공**
- 모바일 환경처럼 보이도록 **User-Agent + 화면 크기 에뮬레이션**
- 즐겨찾기 추가 / 삭제
- 현재 페이지 바로 추가
- 자주 사용하는 사이트 빠르게 접근

---

## 사용 방법

1. 사이드 패널을 엽니다
2. 즐겨찾기를 추가하거나 "현재 페이지 추가"를 클릭합니다
3. 목록에서 원하는 사이트를 클릭합니다
4. 모바일 형태의 팝업 창으로 열립니다

---

## 권한 설명

이 확장 프로그램은 아래 권한을 사용합니다:

- `storage` : 즐겨찾기 저장
- `tabs` : 현재 탭 정보 가져오기
- `debugger` : 모바일 화면 에뮬레이션 (화면 크기 및 User-Agent 변경)

※ 어떠한 사용자 데이터도 수집하거나 외부로 전송하지 않습니다.

---

## 참고 사항

- 일부 웹사이트는 모바일 에뮬레이션을 완전히 지원하지 않을 수 있습니다
- 특정 사이트는 강제로 데스크탑 화면으로 표시될 수 있습니다

---

## 설치 방법

1. 이 저장소를 클론합니다
2. 크롬에서 `chrome://extensions` 접속
3. 개발자 모드 활성화
4. "압축해제된 확장 프로그램을 로드" 클릭
5. 프로젝트 폴더 선택
